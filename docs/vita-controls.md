# PlayStation Vita Controls

This table documents the current Vita-specific behavior.

| Context | Control | Behavior |
|---|---|---|
| Main menu | D-Pad / left analog | Menu navigation |
| Main menu | Cross | Select/confirm (button1); `Y` during char-input prompts |
| Main menu | Start | `Enter` (select/confirm) |
| Main menu | Circle | `Esc` / quit-back |
| Main menu | Triangle | Unused |
| Main menu | Square | Unused |
| Main menu | Select / L / R | Unused |
| World map | D-Pad / left analog | Movement |
| World map | Cross | Jump/button1 + `Enter`; `T` during char-input prompts |
| World map | Start | `Enter` (select/confirm) |
| World map | Circle | `Esc` / quit-back |
| World map | Triangle | Status (`Space`) |
| World map | Square | Fire combo; `D` during char-input prompts |
| World map | Select / L / R | Unused |
| Gameplay (in-level) | D-Pad / left analog | Movement |
| Gameplay (in-level) | Cross | Jump + `Enter` |
| Gameplay (in-level) | Start | `Enter` |
| Gameplay (in-level) | Circle | Pogo |
| Gameplay (in-level) | Triangle | Status (`Space`) |
| Gameplay (in-level) | Square | Fire combo (jump + pogo) |
| Gameplay (in-level) | Select / L / R | Unused |

Notes:
- Prompt aliases are context-specific by design (`Y` in main menu prompts, `T/D` in world map prompts).
- No additional Vita button behavior is intended beyond this table.
